# Game Assets Guide

All games work WITHOUT external assets (using procedural/geometric graphics).
Adding these optional assets enhances the visual quality significantly.

## Source: Kenney.nl (CC0 - Public Domain, free for commercial use)

### Platformer - "Jump Me In"
Download: https://kenney.nl/assets/pixel-platformer
Place files in: `/public/assets/games/platformer/`

Expected files:
- tilemap.png (environment tiles)
- characters.png (player + enemies spritesheet)
- items.png (collectibles, powerups)
- background.png (parallax background)

### Card Game - "Deck Me"
Download: https://kenney.nl/assets/playing-cards-pack
Place files in: `/public/assets/games/cards/`

Expected files:
- cardBacks.png (card back designs)
- cardFaces.png (card face designs)

### Racing - "Fast and Blast" (future)
Download: https://kenney.nl/assets/car-kit
Place files in: `/public/assets/games/racing/`

### Strategy - "Epoch" (future)
Download: https://kenney.nl/assets/board-game-icons
Place files in: `/public/assets/games/strategy/`
